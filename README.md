# TCM Reis — Landing Page (Repaginação)

Protótipos de repaginação do site institucional da **TCM Reis Auditoria | Assessoria**, desenvolvidos pela [Growth Comunicação e Marketing](https://github.com/Growthcomunica) para avaliação do cliente.

## Arquivos

| Arquivo | Descrição |
|---|---|
| `tcm-reis-landing.html` | Versão 1 — identidade "livro-razão": verde-tinta, papel envelhecido, selo de auditoria. |
| `tcm-reis-landing-executivo.html` | Versão 2 — identidade executiva: navy profundo + latão, tipografia serifada, menu mobile e modal de contato. |

## Status

🟡 **Site modelo** — em avaliação com o cliente. As duas versões trazem uma faixa de aviso no topo indicando que o layout ainda não é final.

## Como visualizar

1. Ative o **GitHub Pages** em `Settings → Pages` (branch `main`, pasta raiz).
2. Acesse:
   - `https://growthcomunica.github.io/reistcm/tcm-reis-landing.html`
   - `https://growthcomunica.github.io/reistcm/tcm-reis-landing-executivo.html`

## Conteúdo

Textos, serviços, equipe técnica e dados de contato foram extraídos do site institucional atual ([tcmreis.com.br](http://www.tcmreis.com.br)) e reorganizados em formato de landing page de página única.
